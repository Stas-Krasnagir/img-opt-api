# IMG-conversion

### Single image conversion

http://localhost:5000/api/img/single-img-conversion

**POST**

**Body type:**
_form-data_

```
formData: {
      data : file
    }
```

_return Buffer_

### Multiple images conversion  

http://localhost:5000/api/img/multiple-img-conversion

**POST**

**Body type:**
_form-data_

```
formData: {
      data : [files]
    }
```

_return [Buffer`s]_
